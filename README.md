## Реализовано CRUD для управления профилем пользователя[Laravel]
# Сделано согласно указанным задачам:

- Валидация данных (имя, email, телефон)
- Загрузка аватарки (с обработкой изображений)
- Отображением ошибок валидации
- Успешными сообщениями после сохранения

# При реализации учтены следующие требования:
- Использовать Resource Controller
- Валидация через Form Request
- Обработка изображений через Intervention Image
- AJAX-сохранение без перезагрузки страницы
